Magetool is an open source CLI utility designed to automate common tasks in Magento module development.

Often when you're developing a custom module you find yourself performing certain tasks over and over again, e.g.,

  • Creating a directory structure for the module and activating it
  • Typing out long class names when creating a new controller, block, model, or helper
  • Editing the module's configuration file the first time you create a controller, block, model, or helper

Because these tasks are repetitive and error-prone, they should be automated. Magetool does just that.

See the documentation for a survey of what Magetool can do.

Download and install

Download: magetool-0.1.1.tar.gz

tar xvzf magetool-0.1.1.tar.gz
cd magetool-0.1.1
python setup.py build
sudo python setup.py install

The Magetool source code is hosted on GitHub.

Prerequisites

To run Magetool you need to have the following software installed:

  • Python 2.6
  • lxml

On Ubuntu Linux you can install these dependencies through apt:

sudo apt-get install python2.6 python-lxml

Example usage

Here’s a sample Magetool workflow:

cd ~/public_html/magento/app/code/local/Company
magetool create module HelloWorld                ①
cd HelloWorld
magetool create controller index                 ②
  1. This creates and activates a new shell module called HelloWorld.
  2. This defines a skeleton PHP class in controllers/IndexController.php and updates the module’s configuration file accordingly.
